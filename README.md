
# Sudoku Solver Web Application

🔗 **Live Demo:** https://sudoku-solver-steel-seven.vercel.app/  
💻 **Source Code:** https://github.com/ABHISHEK29BISHNOI/Sudoku-Solver

A simple web-based Sudoku Solver that allows users to enter a 9×9 Sudoku puzzle and automatically solve it.

## Features

- Interactive 9×9 Sudoku board
- User input validation (numbers 1–9 only)
- Automatic puzzle solving
- Clear board functionality

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

1. Enter Sudoku numbers into the grid.
2. Click the **Solve** button.
3. The algorithm processes the puzzle and fills the correct solution.
4. Use **Clear Board** to reset the puzzle.
